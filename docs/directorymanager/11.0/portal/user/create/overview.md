# Create User Objects

GroupID Portal enalbles you to create following types of users:

- User
- Mailbox
- Contact

GroupID supports the following identity providers for creating the user objects:

- Active Directory
- Microsoft Entra ID
- Generic LDAP
- Google Workspace
