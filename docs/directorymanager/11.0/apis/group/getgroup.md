# Get a Group

This Get Group API retrieves details of a specified group.

## Endpoint

**https://demomachine:4443/GroupIDDataService/api/IdentityStores/{identitystoreId}/Groups/{GroupIdentity}**

## HTTP Method

**GET**

#### Sample Request Syntax

```
[
    "displayName",
    "ObjectGuid"
  ]
```
