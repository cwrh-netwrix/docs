# Delete a Route

You can use this API to delete a user-defined workflow route.

:::note
You cannot delete a system workflow.
:::


## Endpoint

**https://demomachine:4443/GroupIDDataService/api/IdentityStores/{identitystoreId}/Workflows/Routes?ids=1**

## HTTP Methods

**DELETE**
