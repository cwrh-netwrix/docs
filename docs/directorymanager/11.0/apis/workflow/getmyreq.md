# Get My Requests

Using this API the logged in user can get information about his/her workflow request(s).

## Endpoint

**https://demomachine:4443/GroupIDDataService/api/IdentityStores/{identityStoreId}/Workflows/MyRequests**

## HTTP Methods

**GET**
