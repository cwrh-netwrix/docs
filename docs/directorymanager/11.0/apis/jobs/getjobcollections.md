# Get Job Collections

Use this API to retrieve information of job collection(s) based on filters provided in the request
syntax.

See the
[Create a Job Collection ](/docs/directorymanager/11.0/portal/synchronize/create/collection/create.md)for
additional information on job collections.

## Endpoint

**https://demomachine:4443/GroupIDDataService/api/DataSyncJobs/Collections**

## HTTP Method

**GET**

#### Sample Request Syntax

```
{
    "filterCriteria": {
        "operator": "is exactly",
        "attribute": "Name",
        "value": "DataSync_API",
        "valueType": 0,
        "valueTypes": [
            0
        ],
        "donotEscapeValue": true,
        "child": [
            {
                "Operator": "string",
                "Attribute": "string",
                "Value": "string",
                "ValueType": 0,
                "ValueTypes": [
                    0
                ],
                "Child": null
            }
        ]
    },
    "SearchOption": {
        "PageSize": 10,
        "PageIndex": 1,
        "SortColumnName": "",
        "SortOrder": 1
    }
}
```
