# Get a Data Source

Using this API you can retrieve information of a specific data source specified in the end point
URL.

## Endpoint

**https://demomachine:4443/GroupIDDataService/api/DataSource/{dataSourceId}**

## HTTP Method

**GET**

#### Sample Request Syntax

```
true
```
