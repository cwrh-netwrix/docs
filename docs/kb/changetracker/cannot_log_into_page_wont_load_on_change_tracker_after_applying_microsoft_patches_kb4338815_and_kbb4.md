# Cannot Log into Change Tracker After Applying Microsoft Patches KB4338815 and KB4338824

## Problem

After updating to KB4338815 and KB4338824, users cannot log into **Change Tracker**. The behavior observed is as follows: users can reach the certificate page; however, when they accept the certificate, the page continues to spin:

![Certificate page spinning after accepting](https://nwxcorp--c.na147.content.force.com/sfc/dist/version/download/?oid=00D7000000091pB&ids=0684u00000LdKUH&d=%2Fa%2F4u000000Lzw7%2FKDetijet.vyLiMFVX7srjP_CTIAH3ye0I5fkGubNBdI&asPdf=false)

## Solution

Patch the system to a version specified to fix issues with these patches as described in the Microsoft roll-up article below:

[Windows 8.1 Update KB4338824](https://support.microsoft.com/en-us/help/4338824/windows-81-update-kb4338824)
