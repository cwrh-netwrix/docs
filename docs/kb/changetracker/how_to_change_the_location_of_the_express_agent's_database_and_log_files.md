# How to Change the Location of the Express Agent's Database and Log Files

## Overview

Change the Express Agent's database and log files with the `ConfigPath` switch in `expressAgent.ini`:

```ini
[ExpressAgent]
ConfigPath=/var/nnt/expressagent/
```
