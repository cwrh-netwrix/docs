# How to Search Archived Data in Log Tracker

## Overview

This article is useful when trying to pull older logs through the web access for **Log Tracker**, but you can only go as far back as 90 days. You can access the `.gz` archive files on the server and unpack them, but you may want to view the message information in the web console for easier data searching. The instructions below will show you how to do that.

## Steps to Take:

1. You will be able to search the **Archive** data from the **Run Query** link as follows:

   ![Search Archive Data from Run Query](https://nwxcorp--c.na147.content.force.com/sfc/dist/version/download/?oid=00D7000000091pB&ids=0684u00000LdKE0&d=%2Fa%2F4u000000LzbZ%2F0X8CR44AMU7.pt11pMR2zDgAsW9FcFiDUzzq_.Z.aKo&asPdf=false)

2. You can then select **Archive** from the dropdown menu for **Query File Type**:

   ![Select Archive from Query File Type](https://nwxcorp--c.na147.content.force.com/sfc/dist/version/download/?oid=00D7000000091pB&ids=0684u00000LdKJw&d=%2Fa%2F4u000000Lzn0%2FaklFjNgLWGIgB97SaJhQdSs3yrZhs0i3zED9yQg5P20&asPdf=false)

3. From there, you will be able to select the file for the date you are looking for:

   ![Select File for Date](https://nwxcorp--c.na147.content.force.com/sfc/dist/version/download/?oid=00D7000000091pB&ids=0684u00000LdJzg&d=%2Fa%2F4u000000LzpF%2FCrqVSmFv1VYf7YhLnDBfJPmWh_Z1WcUo3mlJFm_M3V8&asPdf=false)

4. Set any of the other search parameters you want and then click **Confirm** to run the query:

   ![Click Confirm to Run Query](https://nwxcorp--c.na147.content.force.com/sfc/dist/version/download/?oid=00D7000000091pB&ids=0684u00000LdKK1&d=%2Fa%2F4u000000LzMa%2FPE6PzK8.xOm.2UsmnRc22L69DYDC81bqUJK1YNrePp0&asPdf=false)
